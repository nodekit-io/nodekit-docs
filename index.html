  • Lightweight

    As slim as you want, NodeKit uses the embedded JavaScript engines provided by the operating system

  • Open Source

    Always free and open-source, include NodeKit in commercial and non-commercial projects alike

  • Portable

    The NodeKit specs allow applications to be developed independently of the actual platform on which it runs; 100% (not 99%) of application code reuse across desktop, device and cloud

  • Extensible

    The pattern and reference implementation is particularly well-suited for extending with plugins
